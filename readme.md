# NGIN

NGIN is game engine built on top of the [Heaps Framework](https://heaps.io),
using the [Haxe Programming Language](https://haxe.org) and is shamelessly
inspired by the [HaxeFlixel](https://haxeflixel.com) game engine.

Philosophy
----------

* Minimalistic
* Simple
* Fun
